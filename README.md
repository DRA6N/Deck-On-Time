# Deck on Time

------------

This is a very simple plugin that combines the total uptime since your last SSD install with the total power-on time and gives you a combined overall time you've spent using your Steam Deck.

![](https://i.imgur.com/6BMXmk7.png)

***NOTE: This requires you to have Decky installed***

Simply go into desktop mode, and paste the "deck-on-time" folder into your homebrew folder, which should be located: 

    /home/deck/homebrew

Then go boot back into normal steamOS mode on and use the quick access button (...) > plugins and it should be listed with your other Decky plugins. 

*Disclaimer: I am not responsible if you somehow bork your deck or mess up decky etc. This isn't rocket science but if you haven't ever added any custom plugins/apps before it could be a challenge. Use at your own risk!*
